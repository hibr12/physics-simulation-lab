Interactive Physics Simulation Lab (Forces and Motion)

The Interactive Physics Simulation Lab is a browser‑based tool that demonstrates how forces and motion work in real time. It allows users to add objects, adjust their mass and friction, toggle gravity, and apply external forces. The canvas visualizes each object’s movement with trails, velocity vectors, and force arrows, making the physics concepts easy to see. A control panel provides sliders and buttons for experimentation, while an info panel shows detailed properties of the selected object. At the end of a run, a summary section presents conclusions for each object, including position, velocity, acceleration, and distance traveled.
